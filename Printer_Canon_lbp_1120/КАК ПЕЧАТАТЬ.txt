==================================================================
  ПЕЧАТЬ НА CANON LASER SHOT LBP-1120 (через виртуалку Windows XP)
==================================================================

Зачем так: у принтера нет 64-битного драйвера. Внутри Windows 11
крутится маленькая Windows XP, у неё родной драйвер Canon. Принтер
по USB "отдаётся" внутрь XP, печатаем оттуда. Это запасной принтер,
включаем только когда нужно распечатать.


------------------------------------------------------------------
  РАЗОВАЯ НАСТРОЙКА (делается один раз)
------------------------------------------------------------------
Этот блок выполняется при первом запуске. Если драйвер уже стоит
в XP — пропустите.

1. Скачать драйвер CAPTv102winXP.exe с сайта Canon (страница LBP1120,
   раздел Drivers -> Windows XP).
2. Файл попадёт внутрь XP (помогает Claude / через общую папку или CD).
3. Запустить виртуалку (файл "Печать на Canon.bat").
4. Внутри XP запустить CAPTv102winXP.exe и установить драйвер.
5. Воткнуть принтер в USB, включить его. Если XP попросит драйвер —
   указать уже установленный Canon LBP-1120.


------------------------------------------------------------------
  ОБЫЧНАЯ ПЕЧАТЬ (каждый раз)
------------------------------------------------------------------
1. Включить принтер, воткнуть его USB-кабель в компьютер.
2. Двойной клик по "Печать на Canon.bat" на рабочем столе.
   -> сама запустится виртуалка XP и захватит принтер.
3. В окне виртуалки открыть нужный файл (PDF / документ) и нажать
   Печать, выбрать принтер "Canon LBP-1120".
4. После печати — выключить виртуалку (можно просто закрыть окно ->
   "Сохранить состояние машины") и выключить принтер.


------------------------------------------------------------------
  КАК ЗАКИНУТЬ ФАЙЛ НА ПЕЧАТЬ ВНУТРЬ XP
------------------------------------------------------------------
Вариант А (общая папка): в виртуалке открыть \\vboxsvr\ и взять файл
   из общей папки (если настроена).
Вариант Б: просто перетащить файл мышью в окно виртуалки
   (если установлены Guest Additions с поддержкой drag&drop).
Claude поможет настроить удобный вариант.


------------------------------------------------------------------
  ЕСЛИ ПРИНТЕР НЕ ВИДЕН В XP
------------------------------------------------------------------
В окне виртуалки: меню "Устройства" (Devices) -> "USB" -> поставить
галочку на "Canon ..." . Это вручную пробрасывает принтер внутрь.
Скорость USB 1.1 для печати достаточна.

# Настройка стороны XP (один раз)

Сторож `watcher.vbs` живёт внутри гостевой XP и реально печатает задания.
Ниже — разовая настройка.

## 1. SumatraPDF (XP-совместимая версия)

LBP-1120 печатается из командной строки через SumatraPDF.

- Нужна **версия 3.1.2** — последняя, работающая на Windows XP.
- Прямая ссылка (официальный архив):
  `https://www.sumatrapdfreader.org/dl/rel/3.1.2/SumatraPDF-3.1.2-install.exe`
- Установи в путь по умолчанию: `C:\Program Files\SumatraPDF\SumatraPDF.exe`
  (если иначе — поправь `Const SUMATRA` в начале `watcher.vbs`).

Проверка из командной строки XP:
```
"C:\Program Files\SumatraPDF\SumatraPDF.exe" -print-to "Canon LBP-1120 A4" -silent "\\vboxsvr\Shared\Queue\printed\любой.pdf"
```

## 2. Очереди печати под форматы

Размер бумаги задаём **отдельной очередью на каждый формат** (надёжнее, чем флаги).
Драйвер CAPT уже установлен; добавляем его ещё раз под другими именами.

Для каждого формата (**A4**, **A5**, **B5**):

1. Пуск → Принтеры и факсы → **Установка принтера**.
2. Локальный принтер, **снять** «Автоопределение». Порт — тот же **USB001** (на котором уже работает Canon).
3. Драйвер — **Canon LASER SHOT LBP-1120** (из списка / «Установить с диска» на тот же inf).
4. На вопрос про существующий драйвер — **«Использовать установленный»**.
5. Имя принтера — строго:
   - `Canon LBP-1120 A4`
   - `Canon LBP-1120 A5`
   - `Canon LBP-1120 B5`
6. Общий доступ не нужен. Печать пробной — по желанию.
7. После создания: ПКМ по очереди → **Настройка печати** → выставить её **размер бумаги**
   (A4 / A5 / B5 соответственно) → ОК. Это станет умолчанием очереди.

> Минимум — создать только `Canon LBP-1120 A4`, если другие форматы не нужны.
> Имена должны совпадать с `PRINTER_BASE & " " & paper` из `watcher.vbs`.

## 3. Сторож в автозагрузке

1. Скопируй `watcher.vbs` внутрь XP, например в `C:\CanonBridge\watcher.vbs`.
2. Проверь константы в начале файла (`QUEUE_DIR`, `SUMATRA`, `PRINTER_BASE`).
3. Создай ярлык, который запускает его **без окна** через `wscript`:
   - Цель ярлыка: `wscript.exe "C:\CanonBridge\watcher.vbs"`
4. Положи ярлык в автозагрузку:
   `C:\Documents and Settings\<user>\Главное меню\Программы\Автозагрузка`
   (или Пуск → Все программы → ПКМ по «Автозагрузка» → Открыть).

## 4. Автологин XP (чтобы автозагрузка срабатывала без пароля)

Автозагрузка работает только после входа. Чтобы XP сама входила в систему:

- Пуск → Выполнить → `control userpasswords2` → снять
  «Требовать ввод имени пользователя и пароля» → ОК → указать пользователя/пароль.

## 5. Проверка

1. Запусти VM (можно через кнопку «Запустить принтер (VM)» в приложении или лаунчер).
2. Внутри XP убедись, что процесс `wscript.exe` запущен (Диспетчер задач).
3. На Win11 в приложении выбери PDF, формат A4 → **Печать**.
4. В журнале приложения статусы должны пройти `печать → готово`, лист выходит из принтера.

## Заметки

- **Двусторонняя — только ручная**: печатает нечётные, ждёт переворот стопки,
  печатает чётные. У LBP-1120 нет дуплекс-модуля.
- Логи сторожа отдельно не пишутся; диагностический сигнал — `status\<id>.status.json`
  (его читает и показывает приложение).
- Если задание «зависло» в `awaiting-flip` — приложение покажет диалог; нажми ОК
  после переворота. Таймаут ожидания — 5 минут (`FLIP_TIMEOUT_S`).

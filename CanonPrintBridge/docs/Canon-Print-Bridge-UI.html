Canon Print Bridge · Windows 11 / Fluent

Утилита печати на Canon LBP‑1120

Печать PDF через фоновую виртуальную машину с Windows XP. Одно окно, честные индикаторы готовности, минимум шагов. Ниже — главное окно (предпросмотр скрыт / открыт), окно настроек и раскадровка состояний. Язык интерфейса — русский, светлая тема Fluent.

Легенда индикаторов
Ok — готов
Загрузка — идёт запуск
Off — не запущен
Lost — связь потеряна
Ключевые решения
Одно окно, без вкладок
Сценарий редкий — «изредка напечатать один PDF». Навигация избыточна; всё умещается в один поток. Настройки вынесены в отдельный диалог.
Готовность — всегда на виду
Три индикатора (ВМ / ОС / Принтер) в постоянной карточке. Кнопка «Печать» активна только при VM=Ok, OS=Ok, Printer=Ok и выбранном файле — ошибка «нажал печать, а ВМ не поднята» невозможна.
Предпросмотр — правый сплит
GridSplitter справа, свёрнут по умолчанию, открывается кнопкой. Внутри — встроенный WebView2 (просмотрщик Edge), не своя отрисовка. Для «одного файла изредка» постоянная тяжёлая панель ни к чему.
Журнал и «Дополнительно» — свёрнуты
Технический лог и параметр «Страницы» спрятаны в Expander — по умолчанию свёрнуты, чтобы не шуметь. Ошибки — inline InfoBar, а не модалка. «Завершить работу» — с подтверждением (действие выключает ВМ).

Подтверждение — «Завершить работу» (деструктивно)
Завершить работу?

Фоновая виртуальная машина и Windows XP будут выключены, принтер станет недоступен. Незавершённое задание печати будет прервано. Индикаторы вернутся в исходное (серое) состояние.

↑ «Завершить работу» выключает ВМ и сбрасывает UI в исходное состояние — потому подтверждаем.
